Vue的组件通信方式

2019-05-18 1249

vue组件传值几种方式

1.父组件通过prop向子组件传值

2.子组件通过this.$emit()触发父组组件传递过来的方法向父组件传值

3.兄弟组件之间不能直接传值，需要通过父组件来做间接传值，在这种情况下推荐使用vuex

4.中央事件总线：

//Bus.$emit 发送消息

Bus.$emit(‘inceptMessage’, this.msg)

//Bus.$on 接收消息

Bus.$on(‘inceptMessage’,(msg) => {

​ this.fromComponentAMsg = msg

})
